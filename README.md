# Task Manager

## Prerequisite

- [Node.js v20](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

## Quick Start

Install dependencies:

```sh
yarn install
```

Start development server:

```sh
yarn dev
```

Linting and Styling:

```sh
yarn lint
yarn format
yarn format-check
```

Build production code to `./dist`:

```sh
yarn build
```
